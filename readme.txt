
This is a very  basic project prepared for the CRUD operation in class project.The project requirement document is also shared here.

Maven is used for compile and building of the project.
Below are the commands which needs to be executed on the command prompt by going to the project folder:
1.mvnw clean
2.mvnw install
3.mvnw spring-boot:run

Follow the quick startguide and design document to start with.
